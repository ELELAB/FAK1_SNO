R scripts for the GSNOR analysis


1. TCGA_gene_expresion.r: Get mRNA expression and DEA results for GSNOR
2. TCGA_gene_expression_functions.r: Functions to use in '1. TCGA_gene_expression.r'
3. TCGA_subtype_expression.r: Get mRNA expression and DEA results for TCGA cancer subtypes
4. TCGA_stage_expression.r: Get mRNA expression and DEA results for TCGA tumor stages (based on TCGA clinical information, may not be used)
